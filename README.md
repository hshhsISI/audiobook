# 听书工具（Audiobook Generator）
一个专为新手设计的本地小说听书工具，使用微软 Edge TTS 引擎将文本小说转换为高质量中文语音。

## ✨ 功能特点
- **自动识别小说**：自动读取当前目录下的任意 `.txt` 小说文件，无需修改代码
- **可控生成节奏**：输入 `Y` 生成下一段音频，输入 `Q` 退出程序，完全手动控制
- **固定时长分段**：按每段 5 分钟自动分段，避免生成超大文件，适合长篇小说（如《诡秘之主》）
- **永久保存音频**：生成的音频按序号保存（`第1段.mp3`、`第2段.mp3`...），可随时用其他播放器打开
- **零门槛使用**：纯新手友好，复制代码即可运行，无需复杂配置

## 🚀 快速开始
1.  **安装依赖**
    ```bash
    pip install edge-tts
    ```
2.  **准备小说**
    将你的 `.txt` 格式小说放到项目目录下（文件名不限）
3.  **运行程序**
    ```bash
    python listen_book.py
    ```
4.  **生成音频**
    按照终端提示，输入 `Y` 生成下一段 5 分钟的音频，输入 `Q` 退出程序

## 📝 配置说明
你可以在代码顶部修改以下参数，自定义生成体验：
```python
# 语音选择（二选一）
VOICE = "zh-CN-YunxiNeural"   # 温柔男声
# VOICE = "zh-CN-XiaoxiaoNeural"  # 温柔女声

WORDS_PER_MINUTE = 260  # 每分钟读的字数（默认语速，无需修改）
MINUTES_PER_CHUNK = 5   # 每段音频的时长（分钟），可按需调整
```
